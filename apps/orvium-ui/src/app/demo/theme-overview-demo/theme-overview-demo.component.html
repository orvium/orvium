Chips

Add orv-chip class to mat-chip element. Default font style and opacity would be applied. Afterwards, add any desired color or style class. The classes are located in lib/styles/_chips.scss

  • orv-chip-primary (theme primary color)
  • orv-chip-secondary (theme secondary color)
  • orv-chip-yellow
  • orv-chip-pink
  • orv-chip-blue
  • orv-chip-outlined
  • (disables color change when mouse is on the chip). [disableRipple]="true" is also needed in the chip.
Primary (not clickable) Secondary Yellow Pink Blue Outlined Outlined

If the chips are used to indicate the publication status, the following classes should be used

  • orv-chip-published
  • orv-chip-preprint
  • orv-chip-pending-approval
